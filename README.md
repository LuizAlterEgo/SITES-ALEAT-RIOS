# Abdução Tabacaria e HeadShop - Website

Site profissional desenvolvido para a **Abdução Tabacaria**, referência em Manaus. Foco em experiência do usuário, visual moderno e SEO local.

## 🛠 Tecnologias
- **React** (UI)
- **Tailwind CSS** (Estilização)
- **Framer Motion** (Animações)
- **Lucide Icons** (Ícones)

## Destaques
- **Design Futurista**: Uso de gradientes neon e fundo escuro premium.
- **Conversão**: CTAs estratégicos para WhatsApp em todas as seções.
- **Responsivo**: Adaptado para qualquer tamanho de tela (Mobile-First).
- **SEO Local**: Otimizado para buscas de tabacaria na região da Djalma Batista em Manaus.

## 🚀 Como publicar o site (Passo a Passo)

### 1. Preparação
